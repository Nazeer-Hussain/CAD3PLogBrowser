? Models/CallStackNode.cs (260 lines)
   - Hierarchical call tree node structure
   - Parent/children relationships
   - SelfTimeMs property (excluding child calls)

? Models/PerformanceStatistics.cs (230 lines)
   - Performance metrics for each API method
   - Incremental statistics building
   - Used in Performance tab grid
   - Self-time calculation

? Models/VirtualLogLine.cs (200 lines)
   - Optimized struct for virtual list view
   - Line number, text, background color
   - Factory methods: Create(), CreateNormal(), FromLogEntry()
   - Minimal memory overhead

? Models/SearchResult.cs (230 lines)
   - Find All results data model
   - Match position and length tracking
   - GetHighlightedText(), GetTruncatedText()
   - Used in FindAllResultsForm

? Utilities/Constants.cs (350 lines)
   - Centralizes ~50 magic numbers and strings
   - Organized into logical nested classes:
     • Performance thresholds (Fast/Slow)
     • UI display values (splitter, MRU count)

     • Parsing keywords (ENTER/EXIT, log levels)
     • Application metadata (name, URLs)
     • Color names (error, warning, highlight)
     • Keyboard shortcuts (all shortcuts)
   - Self-documenting constant names

   - Parsing: ToIntOrDefault(), ToLongOrDefault()
   - File: FormatFileSize() (bytes ? "X MB")
   - All methods fully documented with examples
