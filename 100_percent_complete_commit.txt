HOW TO USE:

Export Timeline:
1. Load a log file
2. View Timeline tab (optional, but helps verify data)
3. File ? Export Timeline as Image...
4. Choose filename and format (PNG/JPG/BMP)
5. Click Save
6. High-resolution image (1920x1080) saved!

Export Flame Graph:
1. Load a log file
2. View Flame Graph tab (optional)
3. File ? Export Flame Graph as Image...
4. Choose filename and format (PNG/JPG/BMP)
5. Click Save
6. High-resolution image (1920x1080) saved!

Features:
? PNG format (best quality)
? JPEG format (smaller file size)
? BMP format (uncompressed)
? High resolution (1920x1080)
? Auto-format detection from extension
? Default filename based on log file
? Error handling and validation
? User feedback messages
