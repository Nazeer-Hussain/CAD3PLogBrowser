BONUS FEATURE: Flame Graph Performance Visualization

What is a Flame Graph?
A flame graph is an industry-standard visualization where:
- Width = Time/duration spent in each function
- Height = Call stack depth (hierarchy)
- Color = Function identity (consistent colors)
- Wider bars = Performance hotspots
