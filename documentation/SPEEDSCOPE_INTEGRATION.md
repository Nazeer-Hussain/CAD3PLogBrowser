# Speedscope Integration & Professional Visualization Enhancement

## Overview
Enhanced FlameGraph with **Speedscope export capability** and **speedscope-style vibrant colors** for professional performance profiling.

## What is Speedscope?

**Speedscope** (https://www.speedscope.app/) is an industry-standard, web-based flame graph visualizer used by:
- Chrome DevTools
- Firefox Profiler  
- Node.js profiling tools
- Python profilers
- Many professional development tools

---

## Features Added

### 1. Export to Speedscope JSON Format

Right-click context menu options:
- **Export to Speedscope JSON...** - Save file to share
- **Open in Speedscope.app** - Quick browser launch

### 2. Speedscope-Style Vibrant Colors

16 distinct professional colors (Material Design inspired):
- Red, Pink, Purple, Deep Purple, Indigo, Blue, Light Blue, Cyan
- Teal, Green, Light Green, Lime, Yellow, Amber, Orange, Deep Orange

### 3. Professional Welcome Panels

All three visualizations (Timeline, FlameGraph, CallGraph) now have:
- Clear title and description
- Step-by-step usage instructions
- "Get Started" hints
- Color-coded themes
